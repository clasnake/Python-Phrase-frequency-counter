"# Python-Phrase-frequency-counter"
These scripts count and ranks the frequency keywords and variable # of word phrases in a file/files and generates an html table for display. With pdfkit installed it will then generate a .pdf og that html file.

1) single-docterm-freq.py:
   Counts term frequency for a single file and generate an html table with the results.
2) phrase.py:
   Counts phrase frequency for a single file and generate an html table with the results.
3) multi-file-ngrams.py:
   Counts term frequency for phrases across a specified file type in a given directory for all files of that given type and generates an html table with the results
